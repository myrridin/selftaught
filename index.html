How to be a Self-Taught Programmer

A free 8 week class to learn how to teach yourself

About the Class

This is an 8 week introduction to programming class with an emphasis on teaching and practicing the skill of self development in your craft. It will walk students through the basic concepts of software development, then branch as the class decides to learn interesting technologies and put them to use in live web applications. This is a small class with high interaction and a fast pace, designed for highly motivated students who want to be able to continue advancing on their own after the class is over.


Details

Where:
The Tech Garden

When:
Every Thursday night 6-8PM for 8 weeks starting September 17, 2015

Who:
The ideal candidate is a highly motivated individual with little to no programming experience. The class will help get you through the initial concepts of programming and teach you how to continue learning on your own after the class is over.

How Much:
The first offering of this class will be free. However there is only space for a handful of students. If successful, we'll offer it again as soon as possible.
