How to be a Self-Taught Programmer

A free 8 week class to learn how to teach yourself

About the Class

This is an 8 week introduction to programming class with an emphasis on teaching and practicing the skill of self development in your craft. After a brief introduction to software development concepts, the group will choose an interesting technology and create a live web application. Highly motivated students will discover tools and techniques to advance their software development skills well beyond the classroom.


Details

Where:
The Tech Garden
235 Harrison St
Syracuse, NY 13202

When:
Every Thursday night 6-8PM for 8 weeks starting September 17, 2015

Who:
The ideal candidate is a highly motivated individual with little to no programming experience. The class will help get you through the initial concepts of programming and teach you how to continue learning on your own after the class is over.

How Much:
The first offering of this class will be free. However there is only space for a handful of students. If successful, we'll offer it again as soon as possible.
